Project Overview
This is a Todo CRUD (Create, Read, Update, Delete) web application designed to help users manage daily tasks efficiently. The project focuses on clean UI, responsive design, and a modern dark/light mode toggle for better user experience across devices.

Key Features

Create Todo
Users can add new tasks with details like task title and due date.

Read Todo List
All todos are displayed in a structured list or table with real-time updates.

Update Todo
Existing tasks can be edited easily without reloading the page.

Delete Todo
Users can remove completed or unwanted tasks securely.

Dark Mode / Light Mode

Toggle button to switch between Dark Mode and Light Mode

User-friendly colors that reduce eye strain

Preference can be stored using localStorage for persistence

Responsive Design

Fully responsive layout using Bootstrap / CSS Flexbox / Media Queries

Works smoothly on desktop, tablet, and mobile devices

Security & Best Practices

Form validation on client-side and server-side

Secure database operations using PDO / Prepared Statements

Clean MVC-style structure (optional for scalability)

Tech Stack (Example)

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: PHP

Database: MySQL

Extra: Dark Mode using JavaScript + CSS variables
